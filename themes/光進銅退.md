## 核心驅動力

資料中心機架內互連從銅纜（DAC/AOC）向矽光子 + 光纖轉移，驅動因素為 400G→800G→1.6T 的頻寬升級週期。銅纜在超過 3 公尺後物理特性急劇惡化，AI 超級叢集（GB200 NVL72 等）的大規模部署強制推動光纖化。

## 關鍵里程碑

- [x] 400G 光模組大規模量產，主流雲端完成部署
- [x] 800G 光模組進入規模出貨（2025 H2）
- [x] COHR / LITE 矽光子製程突破，成本持續下降
- [x] 台廠 4977（光環新網）/ 3081（波若威）進入 800G 供應鏈
- [ ] 1.6T 光模組量產（2026 H2 目標）
- [ ] CPO 整合光引擎實現機架內全光化（2027+）
- [ ] 電信骨幹網路升級帶動第二波需求

## 燃料評估

**剩餘燃料：中高（70%）**

800G 尚在爬坡，1.6T 即將接棒，升級週期可見度高。矽光子技術降本推動更廣泛應用。台廠供應鏈逐步受益。主要風險：CPO 加速落地後，外部光模組市場受壓；中國廠商（華為海思光模組）低價競爭。

## 宏觀連結

- 電費上漲 → 節能光模組相對銅纜優勢加大
- 中美脫鉤 → 中國市場份額分流，台美光模組廠受益
