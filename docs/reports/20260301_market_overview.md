## 三、市場新聞與板塊影響

- **美以聯合對伊朗發動大規模空襲** → 影響板塊：能源 / 國防 / 航空 / 科技 → 方向：**能源、國防利多；科技、消費利空**
  > 市場人士警告其市場衝擊「遠大於委內瑞拉事件」，霍爾木茲海峽供應中斷風險使原油供應不確定性驟升，下周油價預期劇烈波動。

- **Anthropic 遭川普政府列黑名單，OpenAI 搶簽五角大廈合約** → 影響板塊：AI 雲端 / 網路安全 → 方向：**OpenAI 合作股利多；Anthropic 相關生態系利空（短期）**
  > 政策面重塑 AI 競爭格局。Anthropic Code Security 工具此前已對網路安全板塊造成「誤殺式」賣壓（直接衝擊 CRWD、NET 等），此次政府表態將強化這一情緒面壓力，但基本面並未改變。🔴 **觀察清單重點關注：CRWD、NET**

- **Meta 與 AMD、Google 簽署數十億美元 AI 晶片多年大單** → 影響板塊：半導體 / AI 基礎設施 → 方向：**利多（AMD、半導體供應鏈）**
  > AMD 將向 Meta 供應高達 6GW 的客製化 AI 晶片，含績效條款。此單正面反映 AI 資本支出維持高強度，且 Meta 正積極多元化供應商（減少 NVDA 單一依賴）。

- **Apple 向台積電（TSM）亞利桑那廠下訂 1 億顆晶片** → 影響板塊：半導體 / 地緣政治製造業 → 方向：**利多（TSM、美國製造概念）**
  > 為目前已知最大宗美國本土晶片採購訂單，強化台積電在川普關稅政策下的在地化布局；同時對 Apple 而言有助於降低供應鏈地緣政治風險。

- **Tesla Cybercab 芝加哥高速公路測試啟動，馬斯克揭示 $30,000 目標售價** → 影響板塊：電動車 / 自動駕駛 → 方向：**利多（Tesla 長期）；短期因馬斯克政治爭議仍有不確定性**
  > Cybercab 量產時程再度成為焦點，但馬斯克本人在 X 上坦承部分川普相關貼文「做得太過火」，市場關注其個人聲譽對 Tesla 品牌的邊際影響。

**近期板塊漲跌背景說明**：科技 / 成長股自 2026 年初持續承壓，納斯達克年初迄今已顯著落後大盤，主要壓力源為聯準會降息步調放緩、AI 商業化兌現時程爭議，以及本周川普政府對 AI 產業的政策干預。能源板塊則因中東局勢升溫而逆勢走強，成為近期市場中少數的防禦性亮點。

## 五、可關注的潛在板塊與龍頭

### 1. 網路安全板塊（AI 誤殺後的結構性反彈機會）

**機會背景**：Anthropic Code Security 工具問世引發的「AI 取代資安」恐慌屬於情緒面過度反應，並非基本面惡化。企業 Zero Trust 架構採用率持續攀升，尤其在地緣衝突加劇的環境下，政府及大型企業資安預算只會增加不會減少。本周 CRWD Q4 財報（3/3）是近期最重要的基本面驗證節點。

**代表個股**：
- 🔴 **CRWD（優先級 1）**：3/3 Q4 財報前最後整理期。ARR $49.2 億、年增 23%，49% 客戶使用 6+ 模組，客戶黏性極強。財報前可小批試探，財報後視 ARR 指引決定是否加倉。目前距 52 週高點 -29%，YTD -17%，估值已明顯修正。
- 🔴 **NET（優先級 2）**：與 Anthropic 工具本無競爭關係，屬嚴重誤殺。Zero Trust + AI 流量管理長期邏輯完整。等技術面帶量止跌 K 棒確認後介入，情緒底部尚未明確。

**切入理由**：雙重利空（Anthropic 誤殺 + 地緣政治 risk-off）疊加造就的錯誤定價，是典型的「基本面未變、情緒面過激」買入窗口。

### 2. AI 基礎設施 / 半導體板塊（資本支出超級周期延續）

**機會背景**：Meta 與 AMD、Google 簽署多年多十億美元大單，以及 Apple 向台積電下訂 1 億顆晶片，共同確認 AI 資本支出週期尚未見頂。NVDA 即將公布 Q4 財報（本周），若數字強勁將成為板塊催化劑。AMD 因 Meta 大單明顯受益，且估值相對 NVDA 折價，風險報酬較優。

**代表個股**：
- **AMD**：Meta 大單 + Nutanix 合作 + AI 推論晶片市占提升，多重利多交匯
- **TSM**：蘋果大訂單 + 美國在地化布局完善 + 近 52 週高點附近整理，基本面強健
- 🔴 **ALAB（優先級 3，觀望）**：Q4 營收年增 92%，Scorpio PCIe 6 全球唯一量產，但 CFO 離職事件技術面尚未止跌，$175 阻力位突破前按兵不動。3/3 Morgan Stanley TMT Conference 或提供新資訊。

**切入理由**：AI 資本支出「超級周期」多空爭議本周因 Meta 大單而暫獲解答，中游基礎設施受益確定性高於純軟體端。

### 3. 能源板塊（地緣衝突驅動的短線交易機會）

**機會背景**：美以聯合空襲伊朗是本周末最大的黑天鵝事件，霍爾木茲海峽若出現任何中斷跡象，原油將面臨劇烈供應衝擊。能源板塊為純防禦性 + 地緣政治溢價的交叉受益者，適合短線戰術性布局，但需嚴設停損（地緣衝突消退速度難以預測）。

**代表個股**：
- **XOM / CVX**：大型整合石油公司，流動性佳，適合快進快出
- **OXY**：巴菲特持倉背書，在原油上行環境中彈性更大
- **VDE**（能源 ETF）：分散單一公司風險的工具性選擇

**切入理由**：純粹的事件驅動交易，以原油期貨走勢為前置指標，設定明確的獲利了結與停損條件，不宜長期持有。

## 六、今日風險提示

### 重大市場風險

- **伊朗局勢升級風險**：美以對伊朗空襲若引發伊朗反制（如封鎖霍爾木茲海峽或發動代理人攻擊），原油可能出現跳空式飆升，全球市場面臨系統性 risk-off 衝擊。此為本周最大尾部風險。

- **周一道瓊期貨開盤跳空**：由於伊朗事件在美股收盤後仍持續發酵，亞洲市場（日本、韓國）周一先行反應，可能導致美股期貨大幅低開，需特別注意流動性風險。

- **AI 政策不確定性**：川普政府將 Anthropic 列黑名單後，市場擔憂這是否為系統性政策轉向（打壓特定 AI 廠商），OpenAI 搶佔政府合約反映 AI 產業政治化風險加劇，可能導致相關股票波動放大。

- **Berkshire 巴菲特效應消退**：Greg Abel 首封股東信雖強調文化延續，但市場需時間消化「巴菲特時代終結」的情緒影響；BRK.B 若出現較大賣壓，可能衝擊對 Berkshire 持倉高度依賴的退休基金配置。

- **Microsoft 估值警訊**：媒體將 MSFT 比擬 Alphabet（GOOGL）的估值陷阱，意味資金可能持續輪出大型科技股，需觀察是否引發 Mag 7 整體的估值重估。
