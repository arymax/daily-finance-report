### 核心業務

Cloudflare 是一家全球網路基礎建設與資安公司，核心產品橫跨三大支柱：

1. **網路效能（Performance）**：CDN（內容傳遞網路）、DNS（1.1.1.1，全球最快公共 DNS）、DDoS 防護、負載均衡、Argo Smart Routing（智能路由）。Cloudflare 自建的全球 Anycast 網路覆蓋 300+ 城市，讓流量就近處理，降低延遲。

2. **Zero Trust 安全（SASE / SSE）**：Cloudflare One 平台整合了 ZTNA（Zero Trust Network Access）、SWG（Secure Web Gateway）、CASB（Cloud Access Security Broker）、DLP（資料外洩防護）、Email Security，以及 2026-03-05 發布的**全球首個完整後量子加密 SASE** 解決方案。這是 NET 最具成長潛力的業務線。

3. **開發者平台（Developer Platform）**：Cloudflare Workers（邊緣運算）、R2（物件儲存，S3 相容，零出口費用）、Pages（前端部署）、D1（邊緣 SQLite）、AI Gateway（AI 推論流量管理）。讓開發者直接在 Cloudflare 網路上部署應用，擴大平台黏性。

**2026 新重點：** AI Gateway 成為 AI 推論流量的過濾與管理層，讓 NET 同時受益於 AI 應用爆發帶來的流量增長與安全需求。

**產品深度補充：**

- **Cloudflare One（SASE 核心）**：以單一控制台統合所有網路與資安功能，客戶無需安裝 on-premise 設備；2026-03-05 宣布整合後量子密鑰封裝（Kyber/ML-KEM 標準），是業界第一個在 SASE 全鏈路部署後量子加密的商業方案，使「harvest now, decrypt later」攻擊對其客戶失效。
- **2026 威脅報告**：NET 每日處理約 4.7 兆次 DNS 請求，等同於全球網路流量的重要樣本，賦予其獨特的威脅情資視野。2026 首份威脅報告揭示 DDoS 攻擊手法向 AI 生成自動化攻擊演進，強化「資安即基礎建設」敘事。
- **AI Gateway**：充當企業 AI API 的統一入口，提供速率限制、日誌記錄、緩存與模型切換功能，讓企業能管控 OpenAI、Anthropic、Gemini 等模型的 API 費用與安全合規。隨企業 AI 採用加速，此產品具備快速規模化潛力。

### 商業模式

- **訂閱制 SaaS**：主要收入來自按計畫（Plan）訂閱，從免費方案升至 Pro、Business、Enterprise。Enterprise 合約通常為年約或多年約，採用量計費（流量、用戶數、座位數）。
- **Land & Expand**：客戶從單一產品（通常是 CDN/DDoS）切入，逐步採用更多模組（Zero Trust、Workers 等），帶動 NRR 持續擴張。Q4 2025 大客戶（$100K+ ARR）數量 YoY 成長超過 27%。
- **平台飛輪**：更多開發者用 Workers → 更多企業依賴基礎設施 → 更多資安需求在同平台解決 → 客戶轉換成本越來越高。
- **免費層作為獲客引擎**：免費方案覆蓋數百萬網站，形成龐大的底層用戶基礎，並驅動口碑傳播與付費轉換。

**收入結構特性：**

- **高度可預測性**：訂閱制為主（Enterprise 多年合約），新書訂單（New Bookings）Q4 2025 年成長約 50%，意味著遞延收入（Deferred Revenue）積累，未來數季收入能見度高。
- **負流失率（Net Revenue Retention）**：NRR 長期維持在 115–125% 區間，代表即使不拓展新客，僅靠現有客戶擴展模組使用就能帶來可觀成長。
- **毛利槓桿**：網路基礎建設具有顯著規模效益，頻寬與硬體折舊成本隨規模攤薄，毛利率中長期仍有提升空間（目前 74.5%，業界對標 Zscaler 約 79%，有追趕空間）。
- **FCF 轉正里程碑**：FY2025 自由現金流利潤率約 10–12%，代表公司已完成「燒錢擴張 → 自給自足」轉型，大幅降低稀釋風險（但需對比新增貨架登記的稀釋量）。

### 客戶輪廓

- **SMB（中小企業）**：透過自助式升級，主要使用 CDN / DDoS 防護 / DNS，ARPU 相對低但規模龐大。
- **中型企業（Mid-Market）**：開始採用 Zero Trust 模組，由 IT 主管或 CISO 主導採購，合約規模 $10K–$100K ARR。
- **大型企業（Enterprise）**：$100K+ ARR 客戶（截至 2025 Q4 約 3,600+ 家），採購決策由 CISO、CTO 及採購委員會共同決定，銷售週期較長（3–9 個月），但合約黏性極高。
- **典型採購情境**：企業 CISO 要整合多個資安點狀工具為統一平台 → Cloudflare One 作為 SASE consolidation 方案 → 替換 Zscaler、Palo Alto 等部分功能，降低複雜度與成本。
- **政府與關鍵基礎設施**：通過 FedRAMP 認證，已有美國政府機構採用，是長期穩定收入來源。

**客戶採購決策鏈深度分析：**

| 客層 | 主要採購者 | 核心考量 | 合約期 |
|------|----------|---------|-------|
| SMB | 技術創辦人 / IT 一人部隊 | 部署速度、無需硬體、免費方案易試用 | 月付或年付 |
| Mid-Market | IT Director / Security Manager | 統一管理介面、合規性、TCO | 1 年 |
| Enterprise | CISO + 採購委員會 | 平台整合廣度、SLA、供應商財務穩健度 | 2–3 年 |
| 政府 | 資訊長辦公室（CIO）+ 採購局 | FedRAMP 認證、資料主權、國安合規 | 多年框架合約 |

**關鍵採購觸發因素：**
- 既有點狀工具合約到期，IT 預算壓縮促使整合採購
- 重大資安事件（如 SolarWinds、Log4j 後浪潮）加速 Zero Trust 部署優先級
- 後量子加密合規期限迫近（美國政府機構預計 2030 年前強制遷移）
- AI 工具大規模部署引發新的資料外洩合規需求（AI Gateway 切入點）

### 競爭護城河

| 護城河類型 | 具體表現 |
|-----------|---------|
| **網路效應** | 覆蓋 300+ 城市、25M+ 網路屬性的 Anycast 網路，規模越大效能越好，新進者難以複製 |
| **資料優勢** | 每天處理 4.7 兆次 DNS 查詢，全球威脅情報的規模讓 AI 安全模型比競爭者更精準 |
| **平台整合性** | 客戶一旦同時使用 CDN + Zero Trust + Workers，更換成本極高（配置複雜、資料遷移難） |
| **後量子加密領先** | 2026-03-05 宣布全球首個完整後量子 SASE，比競爭對手早至少 12–18 個月 |
| **開發者生態** | Cloudflare Workers 社群龐大，形成開發者黏性，間接保護企業端採購 |
| **定價競爭力** | R2 儲存零出口費用，Workers 定價遠低於 AWS Lambda，用成本優勢搶奪開發者市場 |

**主要替代品與競爭對手：**
- CDN / DDoS：Akamai、Fastly、AWS CloudFront
- SASE / Zero Trust：Zscaler（ZS）、Palo Alto Networks（PANW）、Cisco Umbrella
- 邊緣運算：AWS Lambda@Edge、Fastly Compute
- NET 的優勢：**單一平台整合所有功能**，競爭對手通常只擅長一個領域。

**轉換成本量化：**

| 場景 | 轉換障礙 |
|------|---------|
| 從 CDN 遷離 | DNS 重新指向、SSL 憑證遷移、Argo 智能路由配置重建，估計停機風險 2–72 小時 |
| 從 Zero Trust 遷離 | 重新部署 ZTNA 策略、SWG 規則、DLP 設定，涉及全公司員工設備重新配置，通常需 3–6 個月 |
| 從 Workers 遷離 | 重寫邊緣函數至其他平台（AWS Lambda、Fastly），程式碼架構差異大，估計重寫成本達原開發成本 30–50% |
| 整合平台（三者皆用）| 幾乎不可能在短期內遷移，鎖定效應類比 AWS 生態系 |

**後量子先發優勢的戰略意涵：**
NIST 於 2024 年正式發布後量子標準（ML-KEM、ML-DSA），美國 NSA 要求聯邦系統於 2030 年完成遷移。Cloudflare 率先完成全端部署，意味著：
- 政府合約資格提前確立（合規即准入門票）
- 企業客戶若選擇競爭對手，需額外評估後量子遷移路徑與時程
- 技術聲譽加持使銷售週期縮短，尤其在金融與醫療等合規敏感行業

### 成長驅動力（未來 2–3 年）

1. **SASE 整合浪潮**：企業正在把 8–15 個點狀資安工具整合為 1–2 個平台，Cloudflare One 是最具整合深度的候選者。Gartner 預測 2027 年 SASE 市場將達 $25B+，NET 目前僅滲透 ~1–2%。

2. **AI 流量爆炸 → 基礎建設需求**：AI 推論流量（API 呼叫、模型服務）規模快速成長，所有流量都需要 CDN 加速、DDoS 防護與安全過濾。AI Gateway 讓 NET 成為 AI 時代的網路「收費站」。

3. **後量子加密強制化**：美國 NIST 已於 2024 年發布後量子標準，預計 2028–2030 年企業將被強制升級。Cloudflare 是全球最早部署後量子加密的商業平台，2026-03-05 宣布完整後量子 SASE 後，將率先吃下合規驅動需求。

4. **Workers / 邊緣運算市場**：開發者平台 TAM 估計 $100B+（含運算、儲存、資料庫），NET 的邊緣原生架構在低延遲 AI 推論場景具結構性優勢。

5. **大客戶 Land & Expand 加速**：$100K+ ARR 客戶 YoY +27%，新書訂單成長約 50%，意味著未來 4–6 季的收入能見度提升，支撐成長率維持 30%+。

6. **CFO 信心信號（2026-03-05）**：摩根士丹利 TMT 大會上，CFO Thomas Seifert 公開強調強勁成長與 AI 佈局，為市場情緒提供額外支撐。
