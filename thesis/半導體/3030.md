# 德律科技（3030）｜買入依據背景

**產業：** 半導體 / PCB 檢測設備
**市場：** 台灣上市（股票代號：3030）
**最後更新：** 2026-03-05

---

### 核心業務

德律在 PCB 檢測領域提供一條龍服務，涵蓋電路板從研發到量產的完整檢測流程：

- **缺件檢測 → 電測（ICT）→ bump 下的 AXI → Reflow 後 AOI**
- 台灣本土 PCB 檢測設備的主導廠商，客戶轉換成本高，競爭壁壘強

**主力產品線：**

| 產品類別 | 說明 |
|----------|------|
| ICT（In-Circuit Test）| 電路板電氣功能測試，最核心的傳統強項 |
| AOI（自動光學檢測）| Reflow 後焊點外觀缺陷偵測 |
| AXI（X-ray 自動檢測）| 用於 BGA、CoB、覆晶封裝等不可見焊點；半導體封測新戰場的核心 |
| Flying Probe（飛針測試）| 無需治具的樣品 / 少量板測試，切入電動車、醫療少量多樣市場 |
| CoWoS / 先進封裝 X-ray | 新一代半導體設備，支援 wafer bump、TSV 等中段製程 |

---

### 商業模式

德律以**硬體設備銷售為主體**，搭配長期售後服務創造黏性，形成「設備平台 + 服務鎖定」的雙重模式：

1. **設備本體銷售**：一次性高單價（ICT 設備數百萬至千萬新台幣），認列時點為出貨驗收，造成季度間收入波動
2. **耗材 + 維護服務**：探針、治具、年度保養合約，提供穩定持續性收入
3. **軟體授權 / 應用支援**：測試程式（Test Program）開發支援，深度嵌入客戶生產流程
4. **客製化整合方案**：針對大型客戶（如品牌 ODM）提供生產線整合規劃

> **關鍵特性**：設備採購屬資本支出（CapEx），客戶傾向在景氣高峰大量下單、景氣低谷縮手，導致營收高度連動電子業景氣循環。當前 AI 伺服器與先進封裝擴產週期，正是最有利的需求窗口。

### 客戶輪廓

**客戶分層：**

| 層級 | 典型客群 | 購買動機 |
|------|----------|----------|
| 一線 EMS / ODM | 鴻海、廣達、緯創、英業達 | 大規模量產線自動化，ICT / AXI 大批採購 |
| 半導體封測廠 | 日月光、京元電、力成 | CoWoS / 先進封裝良率把關，X-ray 需求爆發 |
| 品牌代工廠 | 台達電、光寶、仁寶 | 混合機種生產線，需彈性測試解決方案 |
| 中小型 PCB 廠 / 設計服務 | 台灣本土 PCB 廠 | Flying Probe 少量多樣，低治具成本 |
| 車電 / 醫療廠 | 台灣與東南亞車用電子廠 | 高可靠度要求，認證門檻高，一旦導入難以更換 |

**採購決策者**：製造工程部（ME）/ 品保部（QA）主導技術評估，採購部執行；高單價設備通常需副總以上簽核，導入週期長（3～6 個月）。

**地理分布**：台灣本土為大本營，中國大陸（江蘇、廣東）佔重要份額，東南亞（越南、泰國）受製造業轉移帶動持續增長，東北亞（韓國、日本）為半導體設備新市場。

### 競爭護城河

**1. 認證壁壘（最核心）**
通過大客戶認證（尤其是 Nvidia CoWoS 供應鏈的 GG 認證）需耗費 12～18 個月，認證一旦取得，非重大品質事故不會更換。德律已完成此關鍵里程碑。

**2. 測試程式生態系（Test Program Lock-in）**
德律設備上運行的測試程式由工程師累積多年開發，檔案格式與設備深度綁定。更換設備等於重寫所有測試程式，沉沒成本極高。

**3. 本土服務響應速度**
台灣本土設備廠的最大優勢是在地服務：半天到場維修 vs. 進口設備等待零件 1～2 週。對於量產線每小時的停機成本，這是不可妥協的議題。

**4. 一條龍整合能力**
單一供應商能涵蓋 ICT + AOI + AXI 全流程，減少客戶管理多家供應商的複雜度，系統整合能力形成差異化。

**5. 技術持續迭代**
R&D 費用率維持在 8～10% 以上，持續開發次世代產品（如 AI 輔助缺陷分類、3D X-ray），確保技術不落後日系（Yamaha、Omron）或德系（Viscom）競爭對手。

### 市場地位

**PCB 檢測設備（台灣）：**
- 德律為台灣 ICT 設備**絕對龍頭**，國內市佔率估計超過 70%
- 全球市場對標：日系（Yamaha Motor、Omron）、美系（Teradyne 旗下品牌）、德系（Viscom）為主要海外競爭對手；德律在台灣市場的在地優勢使其幾乎無可撼動

### 成長驅動力（2026～2028）

**引擎一：AI 伺服器供應鏈的 PCB 複雜度升級**
- AI 伺服器主板層數從 16 層提升至 32 層以上，每片板的測試點數量倍增
- 現有客戶（廣達、緯創等 AI 伺服器 ODM）單台設備需求量與測試時間雙雙增加
- **影響**：PCB 本業在不增加客戶的情況下，自然推動設備汰換與加購

**引擎二：CoWoS / 先進封裝 X-ray 設備放量**
- Nvidia GB200/GB300 需求帶動台灣封測廠瘋狂擴產
- 德律 X-ray 設備從試產導入（~30 台）進入量產規模（目標 ~220 台）
- 單台 ASP（平均售價）顯著高於 PCB ICT 設備，推動整體毛利率上行
- **Re-rating 觸發點**：半導體設備佔比突破 30%，估值邏輯從 PCB 廠升格為半導體設備廠

**引擎三：車用電子滲透率提升**
- 電動車 BMS、ADAS 電路板對測試可靠度要求極高（ISO 26262 車規認證）
- 一旦進入車電供應鏈，換機成本更高、合約週期更長（通常 3～5 年）
- Flying Probe 系列在少量多樣的車電場景具備天然優勢

**引擎四：東南亞製造業移轉效益**
- 越南、泰國、馬來西亞的電子廠擴產，帶動當地 PCB 測試設備需求
- 德律已有海外據點，複製台灣市場經驗向東南亞滲透

### 主要風險

- **營收認列滯後性高**：從 NV 訂規格 → 出貨 → Pilot run → 驗證，流程超過 150 天，季報表現易低於市場預期
- **估值重估時程不確定**：CoWoS 設備佔比提升速度決定 re-rating 是否能兌現
- **近期股價受 ETF 法人調節壓制**，需等技術面確認（W 底）後再評估加碼
- **景氣循環風險**：若 AI 伺服器資本支出意外緊縮，下游 ODM / 封測廠縮減 CapEx，設備採購將快速收縮
- **集中度風險**：對 AI 伺服器供應鏈高度依賴，若 Nvidia 佔比過高，客戶端任何產品延遲均可能影響短期業績
