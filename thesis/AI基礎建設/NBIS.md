# Nebius Group N.V.（NBIS）｜研究報告

**產業：** AI 雲端基礎設施 / Neocloud GPU 算力租用
**市場：** US
**主題：** Nvidia 欽點的歐洲 Neocloud 龍頭——獨立 GPU 雲的結構性稀缺標的，5GW 算力藍圖獲 $2B 戰略背書
**研究日期：** 2026-03-12
**研究觸發：** Nvidia 宣布投資 20 億美元（繼 CoreWeave 後同等級布局），成為 Neocloud AI 算力租用雲的核心受資對象；Nvidia 的戰略性直接投資是強烈的需求側背書，AI GPU 雲端算力的獨立供應商在 Hyperscaler 自建能力有限的細分市場具備結構性成長空間

### 核心業務與商業模式

#### 公司背景：從 Yandex 蛻變的 AI 基礎設施公司

Nebius Group 的前身是 **Yandex N.V.**——俄羅斯最大科技集團的荷蘭上市控股公司。2022 年俄烏戰爭爆發後，Yandex 創辦人 Arkady Volozh 主導一場複雜的企業分拆：將俄羅斯境內業務以約 5,000 億盧布（～$55 億美元）出售，並保留所有**非俄羅斯資產與技術人才**，於 2024 年正式以 **Nebius Group** 名義重新上市（Nasdaq: NBIS），總部登記於荷蘭阿姆斯特丹。

這意味著 Nebius 帶著 Yandex 近 30 年積累的世界頂尖 ML 工程師基因，以一張近乎白紙的資產負債表重新出發——但核心人才、技術能力與 AI 基礎設施的全球野心完整保留。

#### 賣什麼：GPU 雲算力 + AI 開發平台生態

三層產品組合：

**1. Nebius AI Cloud（核心算力）**
- GPU 叢集租用（H100 / H200 / GB200 NVL72）
- 大規模分散式 AI 訓練基礎設施（千卡以上，InfiniBand 互聯）
- AI 推論端點服務（支援主流開源模型）
- GPU-optimized 高速 NVMe 儲存

**2. Nebius AI Studio（開發者平台）**
- 模型即服務（MaaS）API：主流 LLM / 視覺模型 API
- Fine-tuning 服務：企業微調自家模型
- 向量資料庫與 RAG 工具鏈

**3. Toloka（數據標註平台）**
- 前 Yandex 旗下全球最大標註平台，為 AI RLHF 數據準備提供端對端服務

**商業模式本質**：重資產建設（買 GPU、建資料中心）→ 以雲端服務出租 → 利用預留合約鎖定收入可見度。毛利率隨填充率（Utilization Rate）提升而快速改善，前期大量 Capex 消耗現金，規模化後轉為高現金流業務。

### 競爭護城河與市場地位

**Nebius 的 5 道護城河：**

**1. Nvidia 戰略背書（2026 年新確立，最重要）**
Nvidia $2B 直接投資不只是財務注資，更包含：優先 GPU 供貨權（GPU 稀缺市場的決定性優勢）、聯合技術開發（硬體 / 軟體 / 叢集管理）、客戶引薦效應。Nvidia 先後投 CoreWeave 和 Nebius，已確立「Nvidia 認可的 Neocloud 雙核心」格局。

**2. 歐洲市場地緣政治護城河**
GDPR + EU AI Act + 各國資料主權要求，使歐洲企業對非歐系雲端供應商存在天然抗拒。Nebius 是荷蘭法人 + 歐洲資料中心，是 CoreWeave 短期難以複製的本地化優勢，在歐洲 AI 市場有 30-40% 的定價溢價空間。

**3. Yandex ML 工程師密度**
世界級的分散式訓練 / 大規模推論工程能力，直接轉化為客戶端更高的實際算力利用率（Effective FLOPS per Dollar）——這是 GPU 雲的核心差異化之一。

**4. 全棧垂直整合**
算力（AI Cloud）→ 工具（AI Studio）→ 數據（Toloka）三層生態，增加客戶黏性與 ARPU，差異於純 IaaS 模式的競爭對手。

**5. 較輕的資本結構**
相比 CoreWeave 大量舉債擴張（估計負債 >$100 億），Nebius 加上 Nvidia $2B 後財務更健康，在利率環境波動時具備更大韌性。

### 成長驅動力

**驅動力一：Nvidia $2B 投資 → GPU 供貨確定性 + 5GW 算力藍圖**

5GW 算力目標：以 GB200 NVL72 每台 120kW 計算，5GW ≈ 4.2 萬台 NVL72 系統，硬體採購規模達千億美元量級。這揭示了雙方協同擴張的真實野心。

**驅動力二：AI 算力的結構性短缺超級週期**

GPU 生產週期 12-18 個月的結構性滯後、Hyperscaler 優先服務自身工作負載，使獨立 AI 公司 / 研究機構 / 中型企業天然流向 Neocloud。預計 2025-2028 年全球 GPU 雲需求 CAGR >50%。

**驅動力三：歐洲 AI 主權市場的結構性機遇**

歐盟 AI Act（2025 年生效）+ GDPR + 各國數位主權政策，形成對非歐系供應商的天然壁壘。歐盟 AI 投資計劃 2025-2030 年超過 €1,000 億，Nebius 是歐洲 Neocloud 的事實龍頭。

**驅動力四：推論算力需求的爆發（長期最大市場）**

AI 訓練是一次性需求；**推論算力（每次 API 呼叫都在消耗）才是持續性的最大市場**。2026-2028 年推論需求預計超越訓練需求，Nebius AI Studio 已提前佈局。

**驅動力五：Toloka 的數據飛輪（市場低估的隱藏資產）**

RLHF 數據是下一代 AI 模型品質的核心競爭力，Toloka 的全球標註員網路是罕見的規模化數據能力，隨 AI 公司數據需求增加而持續增值。

### 主要風險

#### 風險一：重度資本消耗與執行風險（最核心）
-103% 營業利益率代表持續大量燒錢。5GW 目標需要持續融資、複雜的資料中心建設執行。若市場情緒逆轉，高燒錢公司面臨融資成本急升的生存壓力。資料中心工期延誤、成本超支均可能拖累市場預期。

#### 風險二：地緣政治與公司治理歷史包袱
Yandex 前身的俄羅斯關聯仍可能引發：機構 ESG 篩選迴避、美國監管審查（前俄羅斯資產的合規風險）、核心工程師高薪留才造成的股東稀釋。

#### 風險三：算力定價長期下行壓力
AWS（Trainium）、Google（TPU）、Microsoft（Maia）正大力發展自有 AI 芯片，削弱對 Nvidia GPU 的依賴。中期 GPU 供應增加後，每算力定價（$ per FLOP）可能系統性下降，壓縮 Neocloud 的毛利率護城河。目前 Nvidia 投資是對抗此風險的「政治保險」，但長期格局仍需持續觀察。

### 買入邏輯初步評估

> **結論：高確信成長主題，是值得認真研究的核心標的；當前位置需審慎評估進場時機。**
>
> Nebius 罕見地同時具備三個核心要素：**（1）技術能力強**（Yandex ML 基因）、**（2）市場時機對**（AI 算力的結構性短缺超級週期）、**（3）戰略背書硬**（Nvidia $2B 直接投資）。這三點同時成立的公司在整個 AI 基礎設施賽道中屈指可數。
>
> 估值上，$28B 市值在消息面情緒溢價後並不便宜，但成長軌跡一旦兌現，$50-80B 市值不難想像（+78% to +182% 上行空間）。與 CoreWeave 相比，Nebius 具備更廣地理覆蓋、更健康資本結構、更分散客戶基礎，理應享有合理溢價。當前最大不確定性是從「算力承諾」到「規模收入」的執行風險。
